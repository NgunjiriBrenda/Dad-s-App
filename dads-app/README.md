# For Dad 

A heartfelt, interactive web page created to show appreciation for my dad.

##  Features

- **Beautiful Sky Blue Gradient Background** - Animated gradient that smoothly transitions
- **Floating Hearts Animation** - Hearts float up the screen continuously
- **Interactive Sparkles** - Sparkle effects on mouse movement (desktop) and touch (mobile)
- **Responsive Design** - Works perfectly on mobile, tablet, and desktop
- **Smooth Animations** - Gentle bounce, pulse, and fade-in effects
- **Personal Touch** - Custom photo with glowing border and heart badge

##  Technologies Used

- HTML5
- CSS3 (Animations & Gradients)
- JavaScript (ES6)
- Tailwind CSS (via CDN)

##  How to Use

1. Clone or download this repository
2. Replace `./assets/dad1.jpeg` with your dad's photo
3. Update the link in the "Continue" button to point to your appreciation page
4. Open `index.html` in any modern web browser

## Features Breakdown

### Animations
- Background gradient shifts smoothly
- Hearts float from bottom to top

- Button pulses gently to attract attention
- Sparkles appear on interaction

### Interactive Elements
- Hoverable photo with scale effect

##  Customization

You can easily customize:
- **Colors**: Change the gradient colors in the `body.animate-bg` style
- **Text**: Update the message in the card
- **Photo**: Replace the image path
- **Animations**: Adjust timing in the `@keyframes` sections
